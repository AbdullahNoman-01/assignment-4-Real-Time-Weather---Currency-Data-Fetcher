# 🌦️ Real-Time Weather & Currency Data Fetcher

A simple Python CLI application that fetches real-time weather information and currency exchange rates using public APIs. Users can also save and view the fetched data in a JSON file.

## ✨ Features

- Get current weather by city name
- Get real-time currency exchange rates
- Save fetched data to `data.json`
- View previously saved data
- Simple command-line interface (CLI)

## 🛠️ Technologies

- Python
- Requests
- JSON
- Datetime
- OS

## ▶️ Run

```bash
pip install requests
python data_fetch.py
```

## 🌐 APIs Used

- Weather API: https://wttr.in/
- Currency API: https://open.er-api.com/
